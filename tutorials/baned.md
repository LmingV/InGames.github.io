### 🎨 **符号使用指南**  
1. **颜色编码**（需支持颜色代码）：  
   ```  
   §c⚔️物理伤害 +15  // 红色表示攻击属性  
   §9🛡️物理防御 +20  // 蓝色表示防御属性  
   §d❤️生命力 +50    // 粉色表示生命属性  
   ```  

2. **复合属性显示**：  
   ```  
   ✨🔥 火焰魔法攻击  
   ⚔️+🛠️ 破甲物理伤害  
   ```  

3. **HUD界面整合**（需配合其他插件）：  
   ```  
   [⚔️ 攻击] 245  [🛡️ 防御] 180  
   [❤️ 生命] 1500/1500 [+5📈❤️]  
   ```  

这些符号已在 Minecraft 默认字体中测试可用，若服务器使用自定义字体包，需确保符号兼容性。
